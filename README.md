# Plasmodium

Vite + TypeScript demo app built on `@physarum/client` and `@physarum/server`.

## Dev

```bash
cd /Users/nicholasrogers/Desktop/Coding/plasmodium
npm install
npm run dev
```

- API/WebSocket server: http://127.0.0.1:5177
